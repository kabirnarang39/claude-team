# Anton Example Outputs

These examples are static sample artifacts that show the shape and level of detail Anton workflows aim to produce. They are useful for evaluating the workflow before running Claude Code.

## Examples

- `feature-build-jwt-auth/` - a full feature-build sample for JWT authentication.
- `code-review-sample/` - a compact multi-agent code review sample.
- `bug-fix-failing-test/` - a failing-test triage and fix sample.

For a live UI preview without spending Claude Code tokens, run:

```bash
anton --demo
```

Then open `http://localhost:3000`.
