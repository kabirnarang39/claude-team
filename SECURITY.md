# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| 1.x     | ✅ Yes     |

## Reporting a Vulnerability

**Do not open a public GitHub issue for security vulnerabilities.**

Email: kabir.narang@zinier.com

Include:
- Description of the vulnerability
- Steps to reproduce
- Impact assessment
- Any suggested fix

You'll receive an acknowledgment within 48 hours and a resolution timeline within 7 days.

## Scope

Anton runs as a local server on `127.0.0.1` (localhost only). It is intended for use on a developer's own machine and should not be exposed to the public internet. Running Anton behind a reverse proxy or exposing it on a network interface beyond localhost is outside the supported configuration and is done at your own risk.
